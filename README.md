# Library Project

`library.sh` is a simple shell script designed to manage and organize library-related tasks on Linux/Ubuntu systems.  

---

## Features
- Simple and easy-to-use shell script
- Can run on any Linux/Ubuntu terminal
- Lightweight and fast
- Easily customizable for additional tasks

---

## How to Run
1. Open a terminal in the project directory.
2. Make sure the script has execution permissions:

```bash
chmod +x library.sh
